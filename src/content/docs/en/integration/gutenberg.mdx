The future of content and visual editing in WordPress is Gutenberg. [Gutenberg](https://wordpress.org/gutenberg/) is the new default WordPress editor. It is a block-based editor that allows you to create content with much flexibility.

## Using custom fonts in the Block editor

In the Gutenberg editor, you can use any fonts you have [added](/en/font/custom-font) to Yabe Webfont.

### Step 1: Open the Block setting panel

<a data-fslightbox href="/tutorial/integration/gutenberg/1.png">
![](/tutorial/integration/gutenberg/1.png)
</a>

1. Choose the blocks on the Gutenberg editor's canvas

2. Then click the <code><svg class="inline-block h:1.4em px:2 vertical-align:-.325em fill:current" xmlns="http://www.w3.org/2000/svg" viewBox="0 0 24 24" aria-hidden="true" focusable="false"><path fill-rule="evenodd" clip-rule="evenodd" d="M18 4H6c-1.1 0-2 .9-2 2v12c0 1.1.9 2 2 2h12c1.1 0 2-.9 2-2V6c0-1.1-.9-2-2-2zm-4 14.5H6c-.3 0-.5-.2-.5-.5V6c0-.3.2-.5.5-.5h8v13zm4.5-.5c0 .3-.2.5-.5.5h-2.5v-13H18c.3 0 .5.2.5.5v12z"></path></svg></code> icon on the top right corner of the screen 

3. And switch to the **Block** tab.

### Step 2: Enable the Font Family tools

Click the <code><i class="fa-solid fa-ellipsis-vertical"></i></code> icon on the right of the **Typography** heading title, and click **Font Family** from the dropdown list under the **TOOLS** group to enable the Font Family tools.

<a data-fslightbox href="/tutorial/integration/gutenberg/2.png">
![](/tutorial/integration/gutenberg/2.png)
</a>

### Step 3: Select the font family

Open the **FONT** dropdown list, and select the font family you want to use. The block you have selected will be updated immediately.

<a data-fslightbox href="/tutorial/integration/gutenberg/3.png">
![](/tutorial/integration/gutenberg/3.png)
</a>


## Using custom fonts in the Site Editor

Yabe Webfont also supports custom fonts in the Site Editor, a part of the Gutenberg editor.

### Step 1: Open the Styles setting panel

On the top right of the screen, click the <code><i class="fa-regular fa-circle-half-stroke"></i></code> icon to open the **Styles** setting panel.

<a data-fslightbox href="/tutorial/integration/gutenberg/4.png">
![](/tutorial/integration/gutenberg/4.png)
</a>

### Step 2: Choose the Styles profile

Click the **Browse styles** menu to open the list of styles profiles, and choose the one you want to use.

<a data-fslightbox href="/tutorial/integration/gutenberg/5.png">
![](/tutorial/integration/gutenberg/5.png)
</a>

<a data-fslightbox href="/tutorial/integration/gutenberg/6.png">
![](/tutorial/integration/gutenberg/6.png)
</a>

### Step 3: Typography settings

Click the **Typography** menu to open the typography settings, and choose the element you want to customize.

<a data-fslightbox href="/tutorial/integration/gutenberg/7.png">
![](/tutorial/integration/gutenberg/7.png)
</a>

<a data-fslightbox href="/tutorial/integration/gutenberg/8.png">
![](/tutorial/integration/gutenberg/8.png)
</a>

### Step 4: Select the font family

Open the **FONT** dropdown list, and select the font family you want to use. The element on the canvas will be updated immediately.

<a data-fslightbox href="/tutorial/integration/gutenberg/9.png">
![](/tutorial/integration/gutenberg/9.png)
</a>
