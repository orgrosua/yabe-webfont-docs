Since the inception of WordPress, the [Classic Editor](https://wordpress.org/plugins/classic-editor/) has been the default editor for WordPress. Despite introducing [Gutenberg](/en/integration/gutenberg) as the new default editor, the Classic Editor is still widely used by many WordPress users and developers.

## How to use custom fonts in the Classic Editor

In the Classic Editor, you can use any fonts you have [added](/en/font/custom-font) to Yabe Webfont.
