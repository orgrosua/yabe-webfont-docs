[Zion Builder](https://zionbuilder.io/) is a WordPress plugin that aims to be the fastest, most performant, and most flexible website builder available. It is a tool that allows you to create high-performance websites that load fast both in the back-end and front-end.

## Using custom fonts in the Zion Builder

Yabe Webfont allows you to use [Custom Font](/en/font/custom-font), embedding [Adobe Fonts](/en/font/adobe-fonts) or [self-hosting Google Fonts](/en/font/google-fonts) in the Zion Builder with less effort.

### Step 1: Choose the text element

Choose the text element by clicking on the canvas or the Tree view panel.

<a data-fslightbox href="/tutorial/integration/zion-builder/1.png">
![](/tutorial/integration/zion-builder/1.png)
</a>

### Step 2: Switch to the Styling tab

Click on the Styling tab to switch to the Styling tab.

<a data-fslightbox href="/tutorial/integration/zion-builder/2.png">
![](/tutorial/integration/zion-builder/2.png)
</a>

### Step 3: Choose the selector

Choose the selector you want to apply the font to.

<a data-fslightbox href="/tutorial/integration/zion-builder/3.png">
![](/tutorial/integration/zion-builder/3.png)
</a>

### Step 4: Open the Typography setting

Click the **Typography** menu to open the Typography setting.

<a data-fslightbox href="/tutorial/integration/zion-builder/4.png">
![](/tutorial/integration/zion-builder/4.png)
</a>

### Step 5: Choose the font

Click the **Font Family** dropdown list to open the font list.

<a data-fslightbox href="/tutorial/integration/zion-builder/5.png">
![](/tutorial/integration/zion-builder/5.png)
</a>
